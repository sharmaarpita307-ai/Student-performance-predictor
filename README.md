
# Student Performance Predictor

## About the Project
This project predicts a student's final performance using Machine Learning based on factors such as study hours, attendance, assignments, gender, department, and midterm marks.

## Project Objective
- Prepare and clean the dataset.
- Analyze student performance.
- Train and compare machine learning models.
- Predict final student performance.
